Storage Upgrade HP 245 G8

User complains that 256GB storage isn't enough but doesn't want to change the original NVMe drive due to cost so 2nd best choice is using SATA drive which is cheaper than NVMe drive but faster than even super speed 7200rpm hard drive. Unfortunately it doesn't come with SATA cable or bracket. As a workaround, I've installed a new 256GB SATA drive using bracket and cable from HP 14s since both are compatible.

Memory Upgrade HP 14S-AF118AU

Since Windows 10 21H2 update, running 4GB of RAM in Windows isn't sufficient enough because Microsoft adds more feature that needs more memory. I upgrade this machine to 8GB by adding an SK Hynix 4GB SODIMM DDR3 memory. Sadly the motherboard of this particular laptop is maxed at 8GB but fortunately it's sufficient enough for Windows 10.
